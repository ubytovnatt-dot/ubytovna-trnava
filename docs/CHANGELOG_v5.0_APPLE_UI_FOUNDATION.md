# StayHub v5.0 – Apple UI Foundation

## Cieľ
Prechod zo zložitého admin panelu na jednoduchú workflow aplikáciu pre recepciu.

## Zmeny
- hlavné menu zjednodušené na 4 položky: Dnes, Rezervácie, Check-in / Check-out, Nastavenia,
- nový Apple-style sidebar / mobile bottom navigation,
- nový horný panel s univerzálnym vyhľadávacím poľom ako vizuálny základ,
- Dashboard premenovaný na Dnes,
- doplnené v5 vizuálne tokeny: svetlé sklo, veľké radiusy, jemné tiene, čistejšie karty,
- zachované existujúce funkčné moduly v kóde, ale skryté z hlavného workflow menu.

## Poznámka
Toto je UI foundation. V ďalších krokoch sa budú zjednodušovať pracovné obrazovky Rezervácie a Check-in.
