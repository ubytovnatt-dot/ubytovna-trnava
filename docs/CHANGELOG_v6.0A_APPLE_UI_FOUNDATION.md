# StayHub v6.0A – Apple UI Foundation

## Zmeny

- Nový mobile-first shell `stayhub-v6`.
- Mobilná spodná navigácia: Dnes, Rezervácie, Check-in, Platby, Viac.
- Desktop sidebar prerobený do svetlého Apple štýlu.
- Užšie, ľahšie karty a jemnejšie tiene.
- Platby sú dostupné priamo v hlavnej navigácii pre roly, ktoré ich môžu používať.
- Bez novej Supabase migrácie.

## Nasadenie

1. Nahrať ZIP / push na GitHub.
2. Deploy na Vercel.
3. Supabase netreba meniť.
