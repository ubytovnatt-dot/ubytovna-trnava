# StayHub v3.5A.2.1 – Aurora Timeline Occupancy Days

## Oprava
- Timeline teraz označuje každý deň podľa skutočnej obsadenosti lôžka.
- Denné bunky majú vlastné stavy: voľné, rezervované, ubytované, príchod, odchod, konflikt.
- Opravené vyhodnotenie konfliktu pri kombinácii check-in osoby a rezervácie na rovnakom lôžku/deň.

## UI
- Pridaná jemná denná vrstva pod časovými blokmi.
- Príchod a odchod majú samostatné vizuálne označenie.
- Dnešný deň je zvýraznený rámikom.

## Poznámka
Toto je stabilizačný patch pred pokračovaním na plnohodnotnú Interactive Timeline / Drag & Drop.
