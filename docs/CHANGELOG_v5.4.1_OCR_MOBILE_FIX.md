# StayHub v5.4.1 – AI OCR + Mobile Menu Fix

## Opravené
- Po nahratí dokladu v Check-in je dostupný link **Skontrolovať fotku**.
- OCR používa nahraný obrázok alebo lokálny preview link.
- Ak chýba `OPENAI_API_KEY`, backend vráti jasnú chybu namiesto tichého demo výsledku.
- PDF OCR je zablokované s jasným vysvetlením – pre OCR treba nahrať fotografiu dokladu.
- Mobilné menu je prerobené na full-screen Apple menu so všetkými hlavnými položkami.

## Poznámka
Pre reálne AI OCR musí byť vo Verceli nastavené `OPENAI_API_KEY`.
