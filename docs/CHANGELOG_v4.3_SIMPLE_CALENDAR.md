# StayHub v4.3 – Simple Calendar

## Zmeny
- Zjednodušený modul Kalendár pre recepciu.
- Odstránená komplikovaná Aurora Timeline vrstva z hlavného UI.
- Nové pohľady: Deň, 7 dní, Mesiac.
- Opravený prepínač mesiacov späť / dopredu.
- Prepínač mesiacov používa stabilný výpočet z prvého dňa mesiaca, aby nepreskakoval mesiace.
- Farby zjednodušené: zelená voľné, žltá čiastočne, červená plné, tmavočervená konflikt.
- Build overený cez `npm run build`.
