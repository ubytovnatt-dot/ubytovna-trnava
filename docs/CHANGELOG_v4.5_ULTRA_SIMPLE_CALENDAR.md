# StayHub v4.5 – Ultra Simple Calendar

## Cieľ
Kalendár bol zjednodušený na maximálne praktický recepčný pohľad.

## Zmeny
- odstránené prepínače režimov Deň / 7 dní / Mesiac,
- hlavný pohľad teraz ukazuje naraz:
  - vybraný deň,
  - najbližších 7 dní,
  - zoznam izieb pre vybraný deň,
  - jednoduchý mesačný výber dátumu,
- mesačný prepínač späť/dopredu ostal funkčný,
- menej farieb a menej vizuálneho hluku,
- vhodné pre mobil aj recepciu.

## Farby
- zelená = voľné,
- žltá = čiastočne obsadené,
- červená = plné,
- tmavo červená = konflikt.
