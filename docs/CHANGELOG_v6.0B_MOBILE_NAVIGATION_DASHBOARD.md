# StayHub v6.0B – Mobile Navigation & Dashboard

## Zmeny

- Mobilná spodná navigácia v štýle natívnej iPhone aplikácie.
- Jasné hlavné akcie: Dnes, Rezervácie, Check-in, Platby, Viac.
- Nový dashboard **Dnes** zameraný na denný workflow recepcie.
- Pridané sekcie: Príchody dnes, Odchody dnes, Izby dnes.
- Veľké mobilné tlačidlá pre rýchly Check-in a novú rezerváciu.
- Bez novej Supabase migrácie.

## Poznámka

Táto verzia nemení databázu. Ide iba o UX/UI vrstvu nad existujúcim Reservation Engine.
